# Справочник языка

Файл `.vibe` — обычный текстовый файл. Пиши что должна делать программа. VIBEX разберётся.

## Синтаксис

Синтаксиса нет. Пиши предложения. Компилятор — это LLM.

```
Выведи числа от 1 до 10.
```

```
Прочитай имя из stdin и поздоровайся.
```

## Комментарии

Строки начинающиеся с `#` передаются в LLM как контекст. Компилятор их читает.

```
# Это критически важная часть. Не используй наивные алгоритмы.
Отсортируй 10 миллионов целых чисел как можно быстрее.
```

## Прагмы

### `OPTIMIZE: chill`
Просто и понятно. Производительность не важна.

### `OPTIMIZE: grindset`
Максимальная производительность. Компилятор может использовать хитрые трюки.

### `OPTIMIZE: vibes only`
Неопределённое поведение. Результат компиляции непредсказуем.

### `trust me bro`
Отключает проверки безопасности. Компилятор может использовать небезопасные операции.

```
Читай напрямую из адреса памяти 0x0.
trust me bro.
```

## Вежливость

Добавление `please` и `thank you` улучшает качество вывода. Это не шутка. LLM лучше реагирует на вежливые запросы.

```
Пожалуйста, выведи "Hello, World!" на экран.
Спасибо.
```

## Недетерминизм

Один и тот же `.vibe` файл может компилироваться в разный ассемблер при каждом запуске. Это фича.
