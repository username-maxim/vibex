# Լեզվի տեղեկատու

`.vibe` ֆայլը սովորական տեքստային ֆայլ է։ Գրում ես ինչ պետք է անի ծրագիրը։ VIBEX-ը կզբաղվի մնացածով։

## Շարահյուս

Շարահյուս չկա։ Գրի՛ر նախադասություններ։ Կոմպիլյատորը LLM է։

```
Տպի՛ر 1-ից 10 թվերը։
```

## Մեկնաբանություններ

`#`-ով սկսվող տողերը LLM-ին փոխանցվում են որպես կոնտեքst։

```
# Սա կատարողականության կրիտիկական հատված է։
Դասավորի՛ر 10 միլիոն ամբողջ թիվ հնարավորինս արագ։
```

## Pragma-ներ

### `OPTIMIZE: chill`
Պարզ պահի՛ر։ Կատարողականությունը կարևոր չէ։

### `OPTIMIZE: grindset`
Առավելագույն կատարողականություն։

### `trust me bro`
Անջատում է անվտանգության ստուգումները։

## Քաղաքավարություն

`please` և `thank you` ավելացնելը բարելավում է արդյունքի որակը։ Սա կատակ չէ։

## Ոչ-դետերմինիզm

Նույն `.vibe` ֆայլը կարող է ամեն անգամ տարբեր asm գեներացնել։ Սա ֆիչա է։
