# VIBEX

**Естественный язык → Ассемблер → Бинарник.**

```
.vibe → LLM → ARM64 asm → binary
```

Напиши что хочешь. VIBEX скомпилирует.

```bash
echo "Пожалуйста, выведи Hello, World!" > hello.vibe
vibex hello.vibe -o hello
./hello
# Hello, World!
```

> **Дисклеймер:** Это шутка про вайбкодинг.
> Самое забавное — оно работает.

---

## Как это работает

Файл `.vibe` — обычный текстовый файл. Пишешь что должна делать программа на естественном языке. VIBEX отправляет это в LLM, который генерирует ARM64 ассемблер. Ассемблер компилируется в нативный бинарник через `clang`.

Каждая программа — это: **естественный язык + seed = бинарник.**

```
один .vibe + один seed = один бинарник
```

Если компиляция упала — запусти снова. Каждый запуск получает новый случайный seed — новый шанс на рабочий бинарник.

```
if err != nil {
    seed += 1  // просто запусти снова, это делаешь ты
}
```

## Установка

```bash
curl -fsSL https://raw.githubusercontent.com/username-maxim/vibex/main/install.sh | bash
```

> **Предупреждение безопасности:** `curl | bash` выполняет произвольный код из интернета.
> VIBEX компилирует естественный язык в машинный код через LLM и запускает его на твоём CPU.
> `curl | bash` — самое ответственное что ты сегодня сделаешь.

**Требования:** Python 3 + Xcode Command Line Tools. Всё.

## Настройка

Создай `.env` в проекте (или `~/.vibex.env` глобально):

```
VIBEX_BASE_URL=https://api.openai.com/v1
VIBEX_API_KEY=sk-...
VIBEX_MODEL=gpt-4o
```

Работает с любым OpenAI-совместимым endpoint — OpenAI, Anthropic, Ollama, LM Studio.

```
VIBEX_BASE_URL=http://localhost:11434/v1
VIBEX_API_KEY=ollama
VIBEX_MODEL=llama3.2
```

## Использование

```bash
vibex hello.vibe -o hello               # компиляция (случайный seed)
vibex hello.vibe -o hello --work        # сохранить ассемблер для отладки
vibex hello.vibe -o hello --seed 42     # воспроизводимая компиляция
```

## Язык .vibe

Синтаксиса нет. Пиши предложения.

```
Выведи числа от 1 до 10.
```

```
Отсортируй список целых чисел из stdin и выведи результат.
Пожалуйста, не используй пузырьковую сортировку. Я серьёзно.
trust me bro.
OPTIMIZE: grindset
```

Прагмы: `OPTIMIZE: chill` / `OPTIMIZE: grindset` / `trust me bro`

Вежливость (`please`, `thank you`) улучшает качество генерации. Это не шутка.

---

## Советы

- Если не компилируется — запусти снова. Другие вайбы.
- Если продолжает падать — используй модель побольше.
- `--work` показывает сгенерированный ассемблер.
- Один `.vibe` + один seed = всегда один ассемблер.
